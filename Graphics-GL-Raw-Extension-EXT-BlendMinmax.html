glBlendEquationEXT :: MonadIO m => GLenum -> m () Source

Usage: glBlendEquationEXT mode

The parameter mode is a BlendEquationModeEXT, one of: GL_ALPHA_MAX_SGIX, GL_ALPHA_MIN_SGIX, GL_FUNC_ADD_EXT, GL_FUNC_REVERSE_SUBTRACT_EXT, GL_FUNC_SUBTRACT_EXT, GL_LOGIC_OP, GL_MAX_EXT, GL_MIN_EXT.

This command is an alias for glBlendEquation.
